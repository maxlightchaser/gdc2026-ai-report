# GDC 2026 AI 深度分析报告

可交互批注版报告，支持在线阅读和段落批注。

🔗 **在线查看**: https://maxlightchaser.github.io/gdc2026-ai-report/

## 功能

- 选中文字即可高亮标注（5 种颜色）
- 标注后可附加批注文字
- 右侧面板展示所有标注与批注
- 点击侧边栏卡片定位到文中对应位置
- 支持编辑、删除批注
- 支持导出所有批注内容
- 数据保存在浏览器本地

## 报告内容

- 传统游戏 AI（强化学习、NPC 行为、神经渲染等）
- 生成式 AI（世界生成、3D 资产、引擎整合等）
- 行业生态与从业者态度分析
- 重点厂商技术一览
